jQuery

제이쿼리는 자바스크립트 언어를 간편하게 사용 할 수 있도록 단순화시킨 오픈 소스 기반의 자바스크립트 라이브러리 입니다. 애니메이션과 이벤트 처리를 쉽게 할 수 있으며, 다양한 플러그인이 존재하며 다른 라이브러리와 충돌을 일으키지 않습니다.

link(CDN)

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

제이쿼리를 사용하는 방법은 2가지가 있습니다.
첫번째는 CDN 호스트를 사용하는 방법입니다.
두번째는 직접 제이쿼리를 다운 받아 사용하는 방법입니다.
CDN 호스트를 사용할 경우에는 링크를 적어 주시면 되고, 다운받아서 사용하실 경우에는 경로를 설정하여 적용 시켜 줍니다.

홈페이지 가기

javascript

먼저 자바스크립트로 애니메이션을 설정 하는 방법입니다. 

sliderInner.style.transform = "translateX("+ -800 * currentIndex +"px)";

sliderInner.style.효과 = "translateX( )";

jQuery

다음은 jQuery를 이용해서 애니메이션을 적용 하는 방법 입니다. 

$(".slider__inner").css("position", "relative");
$(".slider__inner").animate({ left : -800 * currentIndex }, 600);

$(선택자).동작함수( );

$(".slider__inner").animate({ left : -800 * currentIndex }, 600);

1. $ 기호는 제이쿼리를 의미를 하며, 제이쿼리에 접근을 하게 해주는 식별자 이기도 합니다.
2. 제이쿼리는 인자값으로 CSS selector를 줄 수 있습니다.
