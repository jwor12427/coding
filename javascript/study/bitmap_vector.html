자바스크립트 기본 규칙

1. 세미콜론(;)

하나의 실행문이 끝나면 세리콜론(;)을 붙여 줍니다. 세미콜론(;)을 붙여 주지 않아도 상관을 없습니다.

var num = 5;
document.write(num);

2. 대소문자

자바스크립트 명령어는 대소문자를 구별합니다. console와 Console은 각각 다른 문자열로 처리 됩니다.

3. 들여쓰기

자바스크립트는 가독성을 위하여 들여쓰기를 사용합니다.

4. 주석

자바스크립트 주석은 일반적으로 특정 코드에 대한 설명을 입력하거나 특정한 부분의 코드가 실행되지 않도록 사용합니다. 주석 표시는 코드에 영향을 미치지 않습니다.

자바스크립트 사용 방법

자바스크립트의 사용방법은 1. 외부파일에 연결하는 방법 2. 스크립트 태그에 사용하는 방법 3. 태그에 직접 기술하는 방법 등이 있습니다. 


    <!DOCTYPE html>
    <html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>Javasript</title>
        <!-- 1. 외부 파일로 로드하는 방법 -->
        <script type="text/Javasript" src="js/script.js"></script>
        <!-- 2. script 태그 사이에 기술하는 방법 -->
        <script>
            document.write("Javasript");
        </script>
    </head>
    <body>
        <!-- 3. 태그에 직접 기술하는 방법 -->
        <a href="#" onclick="alert('Javasript')"></a>
    </body>
    </html>
