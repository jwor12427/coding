레이아웃

레이아웃(layout)이란, 특정 공간에 여러 구성요소를 보기좋게 효과적으로 배치 하는 작업을 의미합니다.

레이아웃 1 : float

float 속성은 해당 HTML 요소가 주변의 다른 요소들과 자연스럽게 어울리도록 만들어 줍니다. 이 속성은 본래 위와 같은 목적으로 만들어졌지만, 현재는 웹 페이지의 레이아웃(layout)을 작성할 때 자주 사용됩니다.

flex 속성은 같은 컨테이너안쪽에 flexible item들에 대한 속성값을 지정합니다. flex의 기본값은 0 1 auto입니다.
