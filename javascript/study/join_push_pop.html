join( ) / push( ) / pop( )

join( ) / push( ) / pop( ) 은 배열 자체를 수정하며, 배열의 처음이나 끝에서 요소를 추가 하거나 제거하는 메서드들 입니다.

join( )

join(` `)

join( )은 배열의 요소를 문자열로 결합하는데 사용됩니다. 

//join()
const arrNum = [100, 200, 300, 400, 500];
const text1 = arrNum.join(``);
const text2 = arrNum.join(` `);
const text3 = arrNum.join(`✨`);
const text4 = arrNum.join(`-`);

push( )

push( )

push( )는 배열의 끝에 하나 이상의 요소를 추가하고, 배열의 새로운 길이를 반환합니다. 

//push()
const arrNum = [100, 200, 300, 400, 500];
const arrPush = arrNum.push(600);
결과보기

이때, arrPush를 출력하면 6이 나오고(배열의 길이), arrNum을 출력하면 배열이 나오게 됩니다.

pop( )

pop( )

pop( )은 배열의 가장 뒤에 있는 요소를 제거 하고 그 요소를 반환합니다. 

//pop()
const arrNum2 = [100, 200, 300, 400, 500];
const arrPush2 = arrNum2.pop();
결과보기

arrPush2는 제거된 값인 500을 반환하고, arrNum2는 마지막 값이 제거된 배열을 출력합니다.
